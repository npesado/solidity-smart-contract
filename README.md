# Project Title

Simple smart contract example

## Description

Smart contract using Rinkeby test network

## Getting Started

### Dependencies

* Docker for devcontainer

### Installing

* npm install

### Executing program

* Run test
```
npm run test
```
* Deploy the contract
```
node deploy.js
```
